## 🏁 Comece

```shell
$ npm install
$ npm start
```

## 📂 Pastas

```
── src
    ├───classes = Classes do projeto
    ├───commands = Funções para todos os comandos da twitch
    ├───handlers = Handlers
    ├───models = Models do mongoose
    ├───routes = Express.js routes
    └───services = Serviços externos (apis)
```
